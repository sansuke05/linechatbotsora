# Line Chatbot "Sora"

自分の所有するドールをモチーフとしたキャラクターLINEボットになります。

現在開発中です。

## 2017年6月現在の開発状況

* ユーザーからのメッセージにランダムで返信する機能を作成。
* ユーザーからのメッセージに画像を一緒に返信する機能を追加。
